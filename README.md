# CareerCompass

CareerCompass is a web-based career guidance platform developed to help students explore suitable career paths based on their skills, interests, and career goals. The platform provides career recommendations, skill guidance, and a user-friendly interface for effective career planning and development.

## Features
- User Registration and Login System
- Career Recommendations
- Skill Gap Analysis
- Dashboard Management
- Responsive Web Design
- Secure Database Connectivity
- Dynamic Web Pages using PHP and MySQL

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Bootstrap

## Project Structure
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Objective
The objective of CareerCompass is to simplify career decision-making by providing students with a centralized platform for career guidance, skill assessment, and career-related resources.

## How to Run the Project
1. Install XAMPP.
2. Place the project folder inside the `htdocs` directory.
3. Start Apache and MySQL services.
4. Import the database into phpMyAdmin.
5. Open the browser and run:
